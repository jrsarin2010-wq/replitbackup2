PROBLEMAS COMUNS E SOLUÇÕES
═══════════════════════════════════════════

❌ "A IA não está respondendo no WhatsApp"
→ Verifique se o WhatsApp está conectado: Configurações → WhatsApp → deve estar verde
→ Se estiver vermelho ou desconectado: clicar em "Reconectar" e escanear o QR Code novamente
→ Verifique se há procedimentos cadastrados (sem procedimentos a IA não funciona bem)

❌ "O WhatsApp desconectou sozinho"
→ Acontece quando o celular fica sem internet ou o app fecha em segundo plano
→ Solução: Configurações → WhatsApp → Reconectar → escanear o QR Code
→ Dica preventiva: manter o celular sempre plugado e com Wi-Fi estável

❌ "A IA agendou no horário errado"
→ Verifique os horários: Configurações → Clínica → Horários de Atendimento
→ Verifique se não há um bloqueio de agenda para aquele período
→ Confirme o fuso horário nas configurações

❌ "Não estou recebendo alertas no Telegram"
→ Verifique se o Telegram está configurado: Configurações → Telegram → deve mostrar "Conectado"
→ Teste clicando em "Enviar teste" — deve chegar uma mensagem no seu Telegram
→ Verifique se o switch de "Alertas" está ativado (deve estar azul/ligado)

❌ "Quero que a IA fale sobre um procedimento específico mas ela não sabe"
→ Cadastre o procedimento: Configurações → Procedimentos → "+ Novo Procedimento"
→ Inclua nome, preço, duração e uma boa descrição para a IA usar

❌ "Novo paciente entrou em contato mas não aparece em Pacientes"
→ Pacientes novos primeiro aparecem em Menu → Leads
→ Após agendar a primeira consulta, migram automaticamente para Menu → Pacientes

❌ "Quero pausar a IA por um tempo"
→ Responda manualmente no WhatsApp — a IA pausa automaticamente por alguns minutos
→ O tempo de pausa pode ser ajustado: Configurações → Telegram → "Tempo de controle manual"

❌ "A recuperação de pacientes não está funcionando"
→ Verifique se está ativada: Configurações → Automação → switch "Recuperação ativa" deve estar ligado
→ Verifique os horários de envio configurados
→ A fila de recuperação aparece em Menu → Recuperação

❌ "O áudio IA parou de funcionar"
→ Verifique se há créditos: Configurações → Áudio IA → saldo de créditos
→ Se zerou, compre mais créditos na mesma tela
→ Verifique se o modo de áudio está ativado (não pode estar em "desativado")
