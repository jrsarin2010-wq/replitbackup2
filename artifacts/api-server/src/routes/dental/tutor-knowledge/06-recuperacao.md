RECUPERAÇÃO DE PACIENTES/LEADS — GUIA COMPLETO
═══════════════════════════════════════════

O que é: módulo que reativa automaticamente pacientes inativos e leads que não compareceram (no-show).

ATIVAR RECUPERAÇÃO:
• Configurações → aba "Automação" → seção "Recuperação"
• Ativar o switch "Recuperação ativa"
• Configurar: dias de inatividade (padrão: 60 dias), horário de envio, mensagem personalizada

QUEM ENTRA NA FILA DE RECUPERAÇÃO:
• Pacientes que não visitam a clínica há X dias (configurável)
• Leads que marcaram e não compareceram (no-show)

COMO FUNCIONA:
• A IA envia mensagens automáticas nos horários configurados
• Monitora respostas e converte em agendamentos automaticamente
• Você vê o status em: Menu → Recuperação (ou Automação)

VER FILA DE RECUPERAÇÃO:
• Menu → Recuperação → lista todos os candidatos com status
• Cada item mostra: nome, último contato, status (pendente/enviado/convertido)

SE A RECUPERAÇÃO ESTIVER PAUSADA:
• Os pacientes acumulam na fila mas não recebem mensagens
• Para reativar: Configurações → Automação → ativar o switch
