ONDE GERENCIAR:
• Menu → Assinatura — vê plano atual, data de contratação, vencimento, status (ativo/cancelado), uso de áudio.

COMO ASSINAR / FAZER UPGRADE / DOWNGRADE:
• Acesse Menu → Assinatura → escolha o novo plano → confirme.
• A cobrança é mensal via PIX (AbacatePay). O sistema gera um QR Code PIX na hora — pague no app do banco.
• Upgrade entra em vigor imediatamente após a confirmação do PIX.
• Downgrade entra em vigor no próximo ciclo (sem reembolso proporcional).

CICLO DE COBRANÇA:
• Cobrança mensal a partir da data de contratação.
• A próxima fatura é gerada automaticamente perto do vencimento.
• Você vê a data exata em Menu → Assinatura → "Vencimento".

CANCELAR ASSINATURA:
• Menu → Assinatura → botão "Cancelar Plano" → confirmar.
• O plano CONTINUA ATIVO até a data de vencimento — você não perde o que pagou.
• Após o vencimento, a conta volta ao plano Básico (ou trial expirado, se nunca pagou).

REATIVAR ASSINATURA:
• Se cancelou e ainda está dentro do prazo: Menu → Assinatura → botão "Reativar".
• Se já venceu: assine de novo escolhendo o plano desejado.

PROFISSIONAL EXTRA:
• R$ 97/mês por profissional além do titular.
• Limite: até 5 extras (6 profissionais no total).
• Cobrança via PIX separada — gera QR Code na hora.
• O profissional fica ativo só após a confirmação do PIX.
• Cancelar um extra: Menu → Assinatura → seção Profissionais Extras → "Cancelar".

CRÉDITOS DE ÁUDIO IA:
• Cada plano pago inclui 20 min de áudio/mês (≈ 20.000 caracteres) que renovam automaticamente.
• 1 minuto ≈ 1.000 caracteres de áudio gerado.
• Quando a cota mensal acaba, o sistema usa créditos de RECARGA (se houver). Quando esses também acabam, a IA volta a responder só por texto.
• Saldo e consumo: Menu → Assinatura → seção "Minutos de Áudio".
