DÚVIDAS TÉCNICAS GERAIS
═══════════════════════════════════════════

QUANDO A IA NÃO RESPONDE:
1. Verifique se o WhatsApp está VERDE em Configurações → WhatsApp. Se não estiver: reconectar.
2. Verifique se há procedimentos cadastrados — sem eles a IA trava em respostas de agendamento.
3. Confira se a IA não está PAUSADA: pausa automática ocorre quando o dentista responde manualmente; aguarde o tempo configurado (Configurações → Telegram → "Tempo de controle manual") ou reabra a conversa.
4. Veja Menu → Conversas em tempo real para confirmar se a mensagem chegou no sistema.

WHATSAPP DESCONECTOU:
• É comum quando o celular fica sem internet, sem bateria ou o app fecha em segundo plano.
• Solução: Configurações → WhatsApp → Reconectar → escanear QR.
• Prevenção: celular plugado na tomada + Wi-Fi estável + não fechar o WhatsApp.

TELEGRAM (TOKEN E BOTFATHER):
• Crie o bot no BotFather (link azul direto na tela) → /newbot → token → cole no OdontoFlow → Validar → /start no bot → "Buscar meu chat" → "Enviar teste".
• Se "Buscar meu chat" não retorna seu nome, é porque você não enviou /start ainda.

ÁUDIO IA (ELEVENLABS):
• Configurações → Áudio IA: ativar/desativar, escolher voz, ouvir preview.
• Modos: "off", "auto" (IA decide), "always" (sempre áudio).
• Consumo: por caractere de áudio gerado. Saldo em Menu → Assinatura.
• Se parou: verifique cota mensal + créditos de recarga.

ANÁLISE DE IMAGEM:
• A IA analisa fotos enviadas pelo paciente (ex: foto do dente).
• Não faz diagnóstico — só descreve e sugere avaliação presencial.
• Não precisa configurar — funciona automaticamente.

CONTROLE MANUAL / PAUSAR A IA:
• Basta o dentista responder no WhatsApp — a IA pausa por X minutos (configurável em Configurações → Telegram).
• Pausa total temporária: pause manualmente em Configurações → Automação (se disponível no plano).

PERSONALIDADE DA IA:
• Configurações → Clínica → "Personalidade da IA": Acolhedora / Profissional / Comercial / instruções customizadas.

FUSO HORÁRIO:
• Configurações → Clínica → "Fuso Horário". Padrão: America/Sao_Paulo.
• Se a IA agendar em horário errado, é quase sempre fuso ou horário de atendimento mal configurado.

PERFORMANCE / SISTEMA LENTO:
• Recarregue a página (Ctrl+F5).
• Limpe o cache do navegador.
• Se persistir e impactar atendimento, abra um chamado pelo Suporte humano (botão "Enviar mensagem para o suporte" na página de Suporte).

NAVEGADORES SUPORTADOS:
• Chrome, Edge, Firefox, Safari (versões atuais).
• Recomendado: Chrome ou Edge.

EXPORTAR DADOS:
• Pacientes, leads e agendamentos podem ser exportados em CSV nas respectivas telas (botão "Exportar" no canto superior).
• LGPD: você pode solicitar exportação completa em Configurações → LGPD.

QUANDO ESCALAR PARA SUPORTE HUMANO:
• Quando o Tutor IA não soube resolver após 2-3 tentativas.
• Erros que não desaparecem após reconectar/recarregar.
• Problemas de cobrança ou pagamento (PIX não confirmado, fatura duplicada).
• Bugs reproduzíveis ou perda de dados.
• Como: página Suporte → seção "2. Enviar mensagem para o suporte".
