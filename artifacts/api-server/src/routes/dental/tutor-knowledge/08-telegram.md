TELEGRAM — GUIA COMPLETO
═══════════════════════════════════════════

O Telegram é usado para receber alertas quando a IA detecta que um paciente precisa de atenção humana (reclamações, emergências, paciente irritado, etc.).

CONFIGURAÇÃO PASSO A PASSO:
1. Configurações → aba "Telegram"
2. Clicar no link azul "abrir o BotFather direto ↗"
3. No BotFather, enviar: /newbot
4. Dar um nome para o bot (ex: "Alertas Clínica Sorriso")
5. Dar um username terminando em "bot" (ex: "ClinicaSorrisoAlertasBot")
6. O BotFather vai enviar um token — copiar o código inteiro
7. Voltar ao OdontoFlow, colar o token no campo indicado
8. Clicar em "Validar"
9. Clicar no link azul que aparece para abrir o bot no Telegram
10. No Telegram, enviar /start para o bot
11. Voltar ao OdontoFlow e clicar em "Buscar meu chat"
12. Selecionar o seu nome na lista
13. Clicar em "Enviar teste" para confirmar

O QUE DISPARA UM ALERTA:
• Paciente pede para falar com um humano
• Reclamações ou insatisfação detectada
• Questões financeiras (reembolso, cobrança)
• Emergências médicas relatadas
• Paciente muito irritado ou agressivo
• Lead recusou agendamento 2 vezes seguidas

TEMPO DE CONTROLE MANUAL:
• Quando o dentista responde, a IA pausa por X minutos
• Configurável em: Configurações → Telegram → "Tempo de controle manual"
• Padrão: 5 minutos. Recomendado: 10-15 minutos para conversas complexas
